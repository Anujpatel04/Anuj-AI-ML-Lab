# Voice-Based Software Interview Coach

Streamlit app that runs a voice-based interview for software roles using OpenAI: STT (Whisper), Chat (evaluation), TTS (questions and feedback). You choose **Company type**, **Role**, and **Difficulty**; the app generates top questions accordingly. Part of Anuj AI ML Lab.

## Features

- **Single-file app** (`app.py`): interview loop, scoring, and audio in one place
- **Company / Role / Difficulty**: choose company type, role, and level; questions are generated to match what companies typically ask
- **Roles**: AI/ML Engineer, Software Developer, Backend, Frontend, Full Stack, DevOps, Data Engineer, Mobile, QA/SDET
- **Company types**: FAANG-style, Big Tech, Startup, Enterprise, Product/SaaS
- **Difficulty**: Beginner, Intermediate, Advanced
- **OpenAI**: STT (Whisper), Chat (evaluation + question generation), TTS (questions and feedback)
- **Config**: `OPENAI_API_KEY` from project root `.env`

## Requirements

- Python 3.10+
- Root `.env` with `OPENAI_API_KEY=your_key_here`

## Setup

```bash
pip install -r requirements.txt
```

Ensure the repo root has a `.env` file with `OPENAI_API_KEY=sk-...`.

## Run

```bash
streamlit run app.py
```

Or from repo root: `streamlit run VOICE_AGENTS/VoiceMLInterview_Coach/app.py`

## Usage

1. Select **Company type**, **Role**, and **Difficulty**
2. Click **Start Interview** (questions are generated for that combination)
3. Listen to each question (TTS), then click **Answer (Record Voice)** and record or upload audio
4. Review transcription, score (0â€“10), and written evaluation
5. Listen to spoken feedback and move to the next question
6. After 5 questions, see final score and overall feedback

## Tech

- **Frontend**: Streamlit; `st.session_state` for company, role, difficulty, questions, answers, scores
- **Questions**: Generated by LLM based on company/role/difficulty (conceptual, practical, reasoning; no coding)
- **STT**: OpenAI Whisper
- **LLM**: OpenAI Chat (gpt-4o-mini) for question generation, scoring, and feedback text
- **TTS**: OpenAI `tts-1` for question and feedback audio
